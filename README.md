# Unit Converter

A responsive static web application that converts values between area, length, temperature, time, volume, and weight units.

This is an individual project created in Semester 1 of the Post Graduate Ontario Certificate in Web Development program at Conestoga College.

## Features

- Category-based unit conversion
- Unit search/filter controls
- Decimal precision and standard/scientific number formatting
- Unit swap and copy-result actions
- Live formula display
- Recent conversion history saved with `localStorage`
- Professional homepage hero image
- Responsive Home, About Us, and Contact Us pages
- Dynamic footer year
- Accessible accordion sections on the About Us page
- Contact form validation with inline status messages

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

Open `index.html` in a browser. No build step or server is required.

## Deployment

This project can be deployed as a static site on GitHub Pages, Netlify, Vercel, or any static hosting service. Upload the full project folder so the `css`, `js`, and `images` paths remain available.
